# Te-Tome — Revolusi Pengelolaan Sampah Digital

Te-Tome adalah sebuah website platform yang membantu masyarakat mengelola dan menukar sampah anorganik (mis. plastik, kertas, kardus, kaleng, kaca) menjadi poin dan reward. Aplikasi ini dibangun dengan React + Vite dan menggunakan TailwindCSS untuk styling.

**Fitur utama**

- **Input Sampah**: Unggah foto/ data sampah untuk memperoleh poin.
- **Daily Poin & Check Poin**: Sistem poin untuk menukarkan reward.
- **Jemput Sampah**: Layanan permintaan penjemputan sampah.
- **Autentikasi sederhana**: Login menggunakan email:user@gmail.com dan pass:user123.

**Judul Website**: Te-Tome

Revolusi Pengelolaan Sampah Digital — mengubah kebiasaan buang sampah menjadi kumpul & tukar sampah dengan reward nyata dan integrasi pengepul daur ulang.

**Prerequisites (Persyaratan)**

- Node.js (disarankan v16 atau lebih baru)
- npm (atau `pnpm`/`yarn` jika Anda prefer, contoh perintah di bawah memakai `npm`)

**Instalasi**

1. Clone repository atau unduh ZIP lalu buka folder project:

```bash
cd path/to/webdesign_Tim-APT_diesnatalis25
```

2. Install dependensi:

```bash
npm install
```

**Cara menjalankan project**

- Jalankan server development (dengan Vite):

```bash
npm run dev
```

    - Server akan berjalan di `http://localhost:5173` secara default (lihat `vite.config.js`). Perintah ini juga membuka browser otomatis.
